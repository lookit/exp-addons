Defined in: addon/randomizers/geometry.js:6
Module: randomizers
Parent Module: exp-player

counterbalance

Object

Defined in addon/randomizers/geometry.js:117

Object describing possible values for counterbalancing conditions.

Sub-properties:

  • startPositions String[]

    list of starting positions for the triangle stream which alternates in both size and shape. (array of strings; options are 'left' and 'right')

  • contextOptions String[]

    list of context triangle options (array of strings; options for context are 'fat', 'skinny')

frameOptions

Object

Defined in addon/randomizers/geometry.js:174

Object describing common parameters to use in EVERY frame created by this randomizer. Parameter names and values are as described in the documentation for the frameType used; see example above.
