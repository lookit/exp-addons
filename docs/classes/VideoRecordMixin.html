recorder

VideoRecorder

Defined in addon/mixins/video-record.js:36

The recorder object. It is the responsibility of the consuming frame to set up the recorder when appropriate, and to set this property. If present, the mixin will automatically use it for things such as capturing stream time.

videoId

String

Defined in addon/mixins/video-record.js:50

A video ID to use for this recording. Defaults to the format <experimentId>_<frameId>_<sessionId>_

There may be additional prefixes/suffixes added elsewhere in the video recording process. A final video captured via this mixin might this have a name like: videoStream_123mcGee_4-phys_eieio5_utctimestamp_random999.flv
